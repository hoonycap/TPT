라이브러리와 프레임워크 (jQuery)

두가지는 비슷하다

라이브러리는 프로그램에 필요한 소프트웨어가 정리되어있어서 뽑아쓰는 컨셉
프레임워크는 프로그램 종류에 따라 공통적인게 이미 만들어져있고, 필요에 따라서 조금씩 다듬어 만드는 컨셉
가장 대표적인 라이브러리 중 하나는 jQuery 이다. //// 다운로드
파일을 다운로드해서 옆에 디렉토리로 직접 붙여넣어도 되고.
사이트에서 CDN(Content Delivery Network)으로 검색해 사용할수도 있다. script src 방식으로 가져갈수있데
Google CDN을 예로 사용해보 / 클릭후 가장 최신인 jQuery 코드를 복사한 다음 / script src 위에 넣어준다. / 그 주소 자체가 이미 script가 되어있는 것이다.
그다음 colors.js로 가서 적용되도록 하면 되는데,
이때 jQuery의 장점은 반복문을 안써도 되도록 해준다는 것이다. 확인하러 가기

jQuery는 '$()' 로 시작한다


따라서 $('a') ----> 모든 a 태그를 jQuery를 이용해 제어하겠다 라는 뜻이다.
jQuery로 다른걸( ex = CSS ) 제어하고 싶다면 'jQuery css' 검색
colors.js 파일에 css('color', color) 는 /// 첫번째 color는 우리가 제어하려는 항목, ////
두번째 color 는 setColor:function(color) 의 매게 변수에 있는 color 값을 입력한 것이다.
삽입한 모든 jQuery 코드를 다른 페이지에도 넣어주면 끝!!!
