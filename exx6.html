객체의 문법


document.write("programmer : "+coworkers.programmer+"br>"); 는 coworkers 라는 객체에서 programer에 해당하는 것을 뽑아오라는 명령어이고
document.write("designer : "+coworkers.designer+"br>"); 는 coworkers 라는 객체에서 designer에 해당하는 것을 뽑아오라는 명령어이다
coworkers.bookkeeper = "duru" 는 coworkers 객체 안에 bookkeeper라는 딱지를 붙인것이 duru 라는 것을 입력하는 거고
document.write("bookkeeper : "+coworkers.bookkeeper+"br>");
를 통해서 bookkeeper 안에 duru 가 들어가 있다는 것을 확인 할 수 있다.
data master 처럼 띄어쓰기가 있는 경우에는
[] 대괄호와 "" 큰따음표를 이용하여, 띄어쓰기를 한 단어로 만들어 줄 수 있다.

Iterate ( 반복문 )


for(var key in coworkers)
document.write(key+'br>')
라는걸 script 안에 넣으면
for 라고 쓰면 coworkers라는 객채가 가리키고 있는 key값을 모조리 무작위 순서로 가져오는것이다 / key는 우리가 원하는 정보의 키 라고 생각하면 간단하다
document.write(key+' : '+coworkers[key]+'br>'); 라고 하면 순서대로 값을 가져온다
여기서 key 값은 programmer / designer / bookkeeper / "data master" 이런것들을 의미한다

Property & Method(객체 안에 함수 담기)


coworkers.showAll = function(){} 을 script에 넣어서 실행하는데 이것과 같은것은
var showAll = function(){} / function showAll(){} 3개가 모두 같은 의미이다.
for(var key in this) 처럼 coworkers 를 this로 바꾸면 상위에 있는 coworkers 가 다른걸로 바뀌어도 그 안에 속해 있는 데이터를 추출해 낼 수 있다.
객체에 소속된 함수coworkers.showAll();를 Method(=coworkers). /// Method 안에서 객체 자신을 가리키는 키워드를 'this' /// 객체에 소속된 변수를 Property 라고 한다.
물론 이렇게 하면 밑에 "showAll" 이라는 부분까지 합쳐져서 나온다는게 문제이다.
